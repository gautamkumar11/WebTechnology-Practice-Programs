 Set() in js collection:

          Syntax of set method:

          var variablename - now set([value1, value2,....valuen])
            methods of set():

            1)add(arg1) - It will take one argument and the argument is value.
                It is used to add new element in the existing set object.

            2)delete(arg1) - It will take one argument and the argument will be value.
                If the data will be present inside set object after deleting the
                data it will return true as output.
                but if the data will not be present it will return false as output.

            3)size - It is used to find the current size of Set.
           
            4)has(arg1) - It accepts one argument and the argument is the value.
                If the value will be present inside Set object it will return true as output.
                If the value will not be present it will return false as output.
            
            5)values() - It is used to fetch the values from the Set object.


            6)entries() - It is used to fetch the key and value both from set Object.
                but inside the set we are not using key.
                so which data we are going to give as value the same data will be 
                considered as key.
                so output will be repeated.

            7)clear() - It is used to clear the Set.
